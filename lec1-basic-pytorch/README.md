# Lecture 1: PyTorch Basics

## Learning Objectives

- Understand the fundamentals of PyTorch
- Learn about tensors and their properties
- Master tensor operations and indexing

## Topics Covered

### 1. Introduction to PyTorch

- What is PyTorch?
- Why PyTorch for deep learning?
- Installation and setup
- Basic imports and environment

### 2. Understanding Tensors

- What are tensors?
- Tensor vs. NumPy arrays
- Creating tensors (zeros, ones, random, from lists)
- Tensor properties (shape, dtype, device)
- Tensor data types (float32, int64, etc.)

### 3. Tensor Operations

- Basic arithmetic operations
- Element-wise operations
- Matrix operations
- Broadcasting
- Common tensor functions (sum, mean, max, min)

### 4. Tensor Indexing

- Basic indexing
- Slicing tensors
- Advanced indexing
- Boolean indexing
- Indexing with tensors
- Modifying tensors in-place

### 5. Tensor Manipulation

- Reshaping tensors (view, reshape)
- Transposing tensors
- Concatenating and stacking tensors
- Splitting tensors
- Squeezing and unsqueezing dimensions

## Key Concepts

- Tensors are the fundamental data structure in PyTorch
- Understanding tensor shapes and dimensions is crucial
- Indexing allows flexible data access and manipulation
